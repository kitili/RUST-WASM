## üöÄ How to Run the Project

### Quick Start
```bash
# Clone repository
git clone https://github.com/kitili/RUST-WASM.git
cd RUST-WASM

# Build WebAssembly module
wasm-pack build --target web --out-dir pkg

# Start local server
python3 -m http.server 8000

# Open http://localhost:8000 in browser
```

### Prerequisites
- Rust (latest stable)
- wasm-pack
- Python 3
- Modern web browser
